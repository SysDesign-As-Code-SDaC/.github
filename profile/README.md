## ğŸš€ Quick Start

### Install the Complete Bundle

```bash
# Clone the master bundle
git clone https://github.com/SysDesign-As-Code-SDaC/sdac-bundle.git
cd sdac-bundle

# Install all components
make install

# Initialize everything
make init-all

# Use the unified CLI
sdac --help
```

### Individual Components

```bash
# Template management
sdac template init --stack backend

# Spec-driven development
sdac spec proposal "Add user authentication"

# Agent protocols
sdac agent protocol secure-code

# Contract testing
sdac contract test --provider pact

# API management
sdac api generate --from-spec

# Security scanning
sdac security scan --fail-on-high
```
